## Descripción

Aplicación de tipo red social cívica que permite a los ciudadanos reportar problemas medioambientales a las autoridades competentes para su seguimiento y resolución. Los usuarios pueden adjuntar fotografías tomadas in situ y la ubicación GPS del problema detectado.

**Funcionalidades principales:**

- Reporte de incidencias medioambientales con foto y geolocalización
- Mapa interactivo con visualización de incidencias
- Vista ciudadano y vista administrador (autoridad pública)
- Registro de usuarios y acceso anónimo con restricciones
- Envío de incidencias por distintos mecanismos
- Panel de administración para gestión y resolución
