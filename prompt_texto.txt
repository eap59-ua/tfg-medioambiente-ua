Una vez completadas las Fases 1-6 (documentación técnica de diseño), el desarrollo del código se realizará en Google Antigravity, el IDE agéntico de Google. Esta sección explica cómo configurarlo y usarlo correctamente para este proyecto.
¿Qué es Google Antigravity?
Google Antigravity (antigravity.google) es un IDE agéntico lanzado en noviembre 2025, basado en VS Code, que permite delegar tareas de desarrollo completas a agentes de IA autónomos. Es gratuito durante la preview pública y disponible para Windows.
Soporta Claude Sonnet 4.6 y Gemini 3 Pro como modelos
Agentes que controlan editor + terminal + navegador simultáneamente
Sistema de Artifacts: planes, capturas, listas de tareas verificables
Tres modos de autonomía configurables por tarea
Base de conocimiento persistente — el agente aprende el estilo del proyecto
